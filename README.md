### HowTo Run:
* go get liokor_mail/cmd/main
* go run liokor_mail/cmd/main

### Другие команды:
* Тесты: go test -v ./...
* Автоформатирование: go fmt liokor_mail/...
* Сборка: go build liokor_mail/cmd/main
